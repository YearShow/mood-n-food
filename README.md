# Mood & Food — МП (прототип)

Это **кликабельный прототип** на мок‑данных (без бэкенда), собранный по фич‑листу от 31.10.2025.

## Как открыть

- Откройте файл `index.html` в браузере:
  - `services/corp/admin/www/prototypes/mood-food/index.html`

## Что реализовано (MVP)

- Авторизация:
  - логин (e-mail) + пароль (мок), переход на главный экран
  - восстановление доступа (мок)
- Главный экран:
  - переходы в “Меню”, “Столы”, “Программа лояльности”
- Меню:
  - категории, список блюд, карточка блюда
  - стоп‑лист: стоп‑блюда нельзя добавить в заказ
- Столы / Заказ:
  - выбор стола
  - добавление гостя (сразу открывает заказ гостя)
  - добавление блюд гостю из меню
  - “заметка для кухни” на позицию
  - “направить заказ” (мок‑успех)
- Программа лояльности:
  - мок‑поиск (список результатов)
  - привязка участника к гостю
  - регистрация участника (создаёт мок‑запись)

## Что специально не делали (post‑MVP)

- статусы столов, бронирование, оплата, пуш‑уведомления, статусы готовности блюд, график смен/отчётность

## Сброс состояния

Прототип сохраняет состояние в `localStorage`. Чтобы начать “с нуля”:
- в браузере очистите localStorage для этого файла/домена, или
- на экране стола нажмите “Сбросить”.
